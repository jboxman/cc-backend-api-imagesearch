Image Search API

API

The API endpoints listed below are available.

GET /api/search?q=<query>[&offset=<number>]

Search for images related to a specific keyword.

Request

A query string is mandatory. An optional offset may be specified. 

$ curl http://localhost:3100/api/search?q=cats
Response

The response is a JSON payload with keys for length, items, and ids.

GET /api/recent

Retrieve a list of the 10 most recent search queries.

Request

No arguments. 

$ curl http://localhost:3100/api/recent
Response

A JSON payload containing an array of objects with query and when keys. The value of when is in UNIX epoch time. 

[
  {
    "query":"cats",
    "when":1476563472926
  },
  {
    "query":"dogs",
    "when":1476559933439
  },
  {
    "query":"dogs",
    "when":1476559933439
  }
]
